Before You Begin

  • Ensure access to Microsoft Excel and the Analysis ToolPak add-in.
  • Familiarity with research design, Excel tools, and basic statistics is recommended.

Lesson Overview

This lesson is for graduate and advanced undergraduate students undertaking original research projects. By the end of this 30-minute lesson, you will be able to format research data, run descriptive statistics, and prepare data for inferential analysis.

Format Data for Analysis

Open the provided Excel document ToolBox_DataLesson.xls and follow these instructions:

  • Ensure variable names are distinct and intuitive.
  • Convert categorical variables to dichotomous (0 or 1).
  • Separate compound variables into single columns.
  • Double-check entries and remove redundant columns.

Tip: Dichotomous Variables

Dichotomous variables (0 or 1) simplify analysis and make identifying correlations straightforward.

Running Descriptive Statistics

  1. Activate Analysis ToolPak in Excel.
  2. Select the data range (excluding labels).
  3. Generate summary statistics.
  4. Name your new worksheet Descriptive_Stats clearly.

Analyzing Results

Descriptive statistics provide insights into central tendencies and data dispersion, helping inform further analysis.

Running Inferential Statistics in PASW (SPSS)

  1. Import formatted data into PASW.
  2. Select appropriate tests (e.g., Kruskal-Wallis, linear regression).
  3. Perform analysis and interpret results.
  4. Export results for documentation.

Choosing the Right Statistic

Refer to UCLA's guide to ensure you're using appropriate statistical methods.
